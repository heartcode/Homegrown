Baseline

To calculate the baseline, we take the default font-size (16px) and multiply it by 1.5. This gives us a baseline of 24px. We then covert these values to em's. This makes the system scaleable.

  • Base font size: 16px (1em)
  • Baseline ratio: 1.5
  • 16px x 1.5 = 24 (1.5em)

Grid System

The grid system is based on the classic 960px layout. The basic system is made up of 60px columns and 20px Gutters. All values are converted to em's

Any combination of columns can be used so long as the total value is no greater than twelve. Column combinations should be used inside a div with a class value of "group". This ensures all floated values are cleared to preserve the grid layout.
